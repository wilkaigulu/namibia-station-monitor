Method

DQS = 0.4·L + 0.4·C + 0.2·G record length relative to a 56-year reference horizon (L), completeness proxy from ISD metadata (C), and coverage of the 1981–2010 climatological normal period (G). Stations at DQS ≥ 0.70 support classical extreme-value estimation. Below that, honest analysis requires prior-informed Bayesian methods with wider credible intervals. Stations confirmed absent from NOAA GHCND receive DQS 0.0 regardless of ISD registration status.

Note: monitor DQS uses ISD metadata as a completeness proxy. For verified scores from actual daily observation counts see the June 2026 audit CSV. Full methodology and station audit: Igulu (2026) · Background reading: wilkaigulu.com.
